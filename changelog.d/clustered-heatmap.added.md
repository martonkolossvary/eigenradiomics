Added `plot_clustered_heatmap`, a generic function that renders a feature-by-feature similarity heatmap with an aligned hierarchical-clustering dendrogram and a per-feature cluster colour strip. It accepts a raw similarity matrix (plus optional cluster labels, linkage, and ordering) or a `ReductionArtifacts` directly, so any reducer's output (e.g. a WGCNA TOM) can be visualized through one interface.
