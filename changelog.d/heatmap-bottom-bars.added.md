`plot_clustered_heatmap` now supports numeric annotation bars below the heatmap via `bottom=` — plain pandas Series (auto-typed, name → title) or `Bar` spec objects for explicit titles/colours. Bars are aligned to the heatmap columns, coloured by module by default (`color="by_module"`, falling back to a single colour when no clusters are given) or any fixed Matplotlib colour, support an optional dashed `reference` line (e.g. −log10(0.05)), and carry the shared feature tick labels on the bottom-most track.
