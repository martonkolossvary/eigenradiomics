`plot_clustered_heatmap` now supports a right-hand correlation panel via `right=` — a features × variables matrix (e.g. features vs clinical variables) drawn with a diverging colourmap (`RdBu_r`, centred at 0) and its own colourbar, aligned to the heatmap's feature rows. Pass a plain DataFrame or a `CorrPanel` spec for explicit colourmap/limits/label. New sourcing helpers `compute_clinical_correlations` (Spearman by default, `RadiomicsDataset`-aware) and `encode_clinical_series` (numeric → binary/ordinal token → alphabetical-ordinal encoding) build that matrix from mixed-type clinical metadata.
