`plot_clustered_heatmap` now supports categorical annotation strips above the heatmap via `top=` — plain pandas Series (auto-typed, name → title) or `Strip` spec objects for explicit titles/colours — plus a stacked right-edge legend column (one titled block per strip and for the module strip) using a colourblind-safe Okabe-Ito default palette. `show_legend` toggles the legend column.
