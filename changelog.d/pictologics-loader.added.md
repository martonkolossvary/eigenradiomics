Added first-class Pictologics ingestion. `RadiomicsDataset.from_pictologics()` loads a wide export (DataFrame or CSV path) in one call — auto-discovering the sidecar `features_catalog.csv`, dropping the `subject_id` leak columns, detecting `config__feature_key` features, and validating them against the catalog (warning on mismatch). `split_observer_tables()` reshapes an observer-paired reproducibility table into one feature matrix per observer for `compute_reproducibility`, and `RadiomicsFeatureRemover` gains an `observer_prefixes` argument so `families=`/`configs=` selectors and catalog joins resolve on the base feature name. `RadiomicsFeatureRemover(catalog=...)` now also accepts a `FeatureCatalog` object (previously only a DataFrame or path).
